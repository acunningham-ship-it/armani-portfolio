# Armani Cunningham — Portfolio

Animation · Photography · Graphic Design. A faithful rebuild of the
[junior Wix portfolio](https://armanicunningham.wixsite.com/junioportfolio) in the
**ARSENIC** layout — same pages, same content, new design + a lightbox gallery system.

**Pages:** Home · Junior Mastery · Animation · Digital Arts 1 · Digital Arts 2 · Portfolio · ACE · Image Editing · Resources · Vocab · Animation Vocab

- `index.html` / `styles.css` / `app.js` — data-driven SPA (hash routing, lightbox)
- `gallery/<page>/` — extracted gallery images, organized by page
- `gallery/manifest.json` — per-page image manifest

Live via GitHub Pages. Open `index.html` locally, or `python3 -m http.server`.
